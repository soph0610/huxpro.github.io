## JAVA 접근 제어자

1. public
: 어디서든 접근 가능한 인스턴스 변수 및 메서드

2. private
:선언된 클래스 내부에서만 접근 가능

3. protected
:클래스 내부, 같은 패키지, 상속받은 클래스에서 접근 가능하게 함.

4. default
: 접근 지시자를 선언하지 않는 것으로, 클래스 내부 및 같은 패키지 내에서 접근 가능하게 함.
